About Me

「一緒に働きたい」と思われる
エンジニアを目指して

私が開発において最も大切にしているのは「協調性」です。単に要件を満たすコードを書くのではなく、後から読むメンバーが理解しやすい構造にすること、仕様の不明点があれば積極的に周囲と認識をすり合わせることを心がけています。

プロジェクトで行き詰まっているメンバーがいれば、自身のタスクを調整してペアプログラミングやレビューを通じてサポートに入ります。「チーム全体の成功」を第一に考え、開発現場の潤滑油として機能することにやりがいを感じています。

Core Values & Skills

協調とサポート

自身のタスク完遂だけでなく、チーム全体のボトルネック解消を優先し、ペアプロ等を通じて柔軟にサポートに入ります。

メンターシップ

ジュニアメンバーの技術的成長を促すため、丁寧なコードレビューや設計相談の時間を積極的に設けています。

心理的安全性

誰もが意見を言いやすく、失敗を恐れずに挑戦できる、風通しの良いアジャイルなチーム文化の醸成を推進します。

フロントエンド & バックエンド

  • React / Vue.js を用いたモダンなUI構築とコンポーネント設計。
  • 再利用性と可読性を重視し、属人化を防ぐコード品質の担保。
  • Node.js / Python を活用し、フロントエンド側が使いやすいAPI設計。

チームビルディング

  • アジャイル開発におけるスクラムマスターとしてのプロジェクト推進。
  • デイリースクラムやレトロスペクティブ等のイベントファシリテーション。
  • 心理的安全性を高めるための、1on1やペアプログラミングの積極的な導入。
